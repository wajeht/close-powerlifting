# Getting Started
Clone the repository

```bash
$ git clone https://github.com/wajeht/close-powerlifting.git
```

Copy `.env.example` to `.env`

```bash
$ cp .env.example .env
```

Install dependencies

```bash
$ npm install
```

Run development server

```bash
$ npm run dev
```

Test the application

```bash
$ npm run test
```
