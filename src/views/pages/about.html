Why?

Within powerlifting world, there is Open Powerlifting ↗️ project exists to create a permanent, accurate, convenient, accessible, open archive of the world's powerlifting data.

They even mentioned not to scrape the website yet still provided bulk data for free of download here ↗️.

Close Powerlifting ↗️ only exists to create reverse API on top of Open Powerlifting ↗️ project for us developers. The primary reason for this project is for education purposes only!

How?

Step 1

Register for api key here ↗️. You will have confirm your email address after registration and API key will sent to your email!

Step 2

Visit our API Documentation ↗️ page to reference all the available routes.

Step 3

Attach your api key via x-api-key or bearer token. 

// auth through x-api-key
const res = await window.fetch('/api/rankings', {
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': <your api key here>,
  },
});
const data = await res.json();
// auth through bearer token
const res = await window.fetch('/api/rankings', {
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer <your api key here>',
  },
});
const data = await res.json();
Enjoy!
