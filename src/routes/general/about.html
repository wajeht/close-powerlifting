A free API that lets developers build apps with the world's largest powerlifting database.

The story

OpenPowerlifting.org is an incredible resource — over 3 million competition results from 100+ federations worldwide, all freely available. But if you want to build an app with that data, you're stuck scraping HTML tables and parsing inconsistent formats.

Close Powerlifting was built to solve that problem. Instead of spending days writing scrapers and handling edge cases, developers can now query the entire database with a simple REST API.

Whether you're building a coaching app, analyzing strength trends, or just want to look up your own competition history programmatically — this API makes it possible in minutes.

Built on OpenPowerlifting

All data is sourced from OpenPowerlifting.org, a community-driven project that archives powerlifting competition results. They do the hard work of collecting and maintaining the data — we just make it easier to access programmatically.

Consider supporting their project if you find this API useful.

Frequently asked questions

Is it free to use?
Yes. Every account gets 500 free API calls per month, which resets at the beginning of each month. No credit card required.
How often is the data updated?
The database syncs with OpenPowerlifting weekly. New competition results are typically available within a few days of being added to their database.
What if I need more API calls?
Contact us with your use case. We're happy to increase limits for legitimate projects.
Can I use this for commercial projects?
This project is primarily educational. For commercial use, please review OpenPowerlifting's data license and consider supporting their project directly.
I lost my API key
Visit the reset API key page and enter your email. A new key will be sent to you.
