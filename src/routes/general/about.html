About

Learn how to get started with the Close Powerlifting API.

This project is intended for educational purposes. Data is sourced from OpenPowerlifting.org, a community-driven archive of powerlifting competition results.

Frequently asked questions

Is Close Powerlifting free to use?
Yes. Every account receives 500 free API calls per month. This resets at the beginning of each month.
How often is the data updated?
Our database syncs with Open Powerlifting weekly. New competition results are typically available within a few days of being added to their database.
What if I need more than 500 API calls?
Contact us through our contact page and let us know your use case. We're happy to increase limits for legitimate projects.
Can I use this for commercial projects?
This project is primarily educational. For commercial use, please review Open Powerlifting's data license and consider supporting their project directly.
Where does the data come from?
All data comes from OpenPowerlifting.org, a community-driven project that archives powerlifting competition results from federations worldwide.
I lost my API key. How do I get a new one?
Visit the reset API key page and enter your email address. A new key will be sent to you.
Is there a way to get real-time competition data?
No. Competition results are added to Open Powerlifting after meets conclude, and we sync weekly. This API is best suited for historical data analysis, not live tracking.
