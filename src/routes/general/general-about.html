This project is intended for educational purposes only.

Why?

Open Powerlifting is a community-driven project that maintains a permanent, accurate, and accessible archive of the world's powerlifting data.

While they request that users avoid scraping their website, they generously provide bulk data for free here.

Close Powerlifting provides a reverse-engineered API on top of Open Powerlifting for developers who want programmatic access to this data.

How?

Step 1

Register for an API key. After registration, you'll need to confirm your email address, and your API key will be sent to you.

Step 2

Visit our API documentation to see all available endpoints.

Step 3

Include your API key in requests using either the x-api-key header or a bearer token. 

// Using x-api-key header
const res = await window.fetch('/api/rankings', {
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': <your api key here>,
  },
});
const data = await res.json();
// Using bearer token
const res = await window.fetch('/api/rankings', {
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer <your api key here>',
  },
});
const data = await res.json();

Each account receives a quota of 500 API calls per month. This limit resets at the beginning of each month. If you need a higher limit, please reach out through our contact page.
