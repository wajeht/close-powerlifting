Flexible queries

/rankings

Filter by federation, weight class, equipment, sex, and year. Sort by total, squat, bench, or deadlift.

/records

World and federation records. Filter by equipment type (raw, wraps, single-ply, multi-ply) and sex.

/meets

Search competitions by federation, country, or date range. Includes full results for each meet.

/users/{'{'}name{'}'}

Complete athlete profiles with competition history, personal records, and progression over time.

Simple integration

// Get top 10 raw male lifters in the 83kg class
const res = await fetch('/api/rankings?equipment=Raw&sex=M&weight_class=83', {
  headers: { 'x-api-key': YOUR_API_KEY }
});

const { data } = await res.json();
// [{ name: "John Haack", total: 907.5, squat: 303, bench: 220, deadlift: 384.5 }, ...]
